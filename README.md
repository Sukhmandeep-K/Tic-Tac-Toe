# Tic-Tac-Toe
Tic-Tac-Toe (Python) A clean, interactive Tic-Tac-Toe game built using Python and the tkinter library. This project features a graphical user interface (GUI), turn-based gameplay, win detection (horizontal, vertical, and diagonal), and a tie-handling system. 🚀 Features Interactive GUI: Built with Python’s built-in Tkinter framework. Smart Win Detection: Automatically identifies and highlights the winning row, column, or diagonal. Reset Functionality: Includes a restart button to clear the board and start a new match instantly. Responsive Feedback: Updates a status label to show whose turn it is or who won the game.
